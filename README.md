# Electric Car Rental System

A desktop booking system for an electric car rental company, built in Python using AppJar and SQLite.

## Overview
This project was created to replace an outdated paper-based booking process with a digital system that reduces booking errors, improves organisation, and makes it easier for customers and staff to manage rentals.

## Features
- User sign up and login
- Password recovery flow
- Vehicle search and filtering
- Booking creation
- Booking summary
- View past and future trips
- Amend bookings
- Cancel bookings
- Favourite cars
- Validation for user input
- Packaged Windows executable

## Technologies Used
- Python
- AppJar
- SQLite / SQL
- PyInstaller

## Why I Built It
The aim of the project was to create a more reliable and user-friendly booking system for an electric car rental business. The original paper-based approach could lead to booking mistakes, poor organisation, and lost information.

## Limitations
- Desktop application only
- No online payments
- No web/mobile version
- Built with AppJar, so UI customisation is limited
